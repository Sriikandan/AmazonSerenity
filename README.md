# AmazonSerenity
Amazon Test using Serenity Cucumber  

Install IntelliJ community Free Version  
Clone this repo  
Open the SRC folder
Execute Maven Goal  
  mvn install  
Once requried JAR files installed
Execute Maven Goal  
  mvn verify  
  
Result is published in the Serenity Report as like below
